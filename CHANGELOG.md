# Changelog

All notable changes to QuestCoin will be documented in this file.

## [1.0.0] - 2026-01-20

### Added
- Initial release of QuestCoin mini-app
- QuestToken ERC20 contract with 1 billion max supply
- QuestHub for quest management and tracking
- QuestVault for reward distribution
- QuestBooster for streak multipliers
- Farcaster Frame integration
- Dark theme UI
- Wallet connection support
- Real-time stats display

### Smart Contracts
- Deployed on Base Mainnet
- OpenZeppelin security patterns
- Pausable functionality
- Emergency withdrawal capability

### Features
- Daily check-in quest
- Engagement tracking
- Commitment verification
- Streak-based boost system
- Referral bonuses
